Fundamental Concepts
Encapsulation
Encapsulation describes the idea of wrapping data and all the methods that work on said data, within one unit.
Inheritance
Inheritance is the capability of one class to derive or inherit properties from another class.
Polymorphism
Polymorphism the word means having many forms. In programming, polymorphism means having the same function name (but different signatures) being used for different types.
Abstraction
Abstract classes can be considered as blueprints for other classes. It allows a set of methods to be created within child classes that are built from the abstract class. A class with one or more abstract methods is an abstract class.

Protected Members
Protected members are the members of a class that exist and can only be accessed only from within a certain class and it's subclasses. In Python, this is accomplished by following a convention of prefixing the name of the member by a single underscore "_".
Private Members
Private members, although similar to protected members, differ in that they are neither accessed outside of their class nor by any base class. In Python, private members can only be accessed inside of a class. Private memebers are defined by a prefix of a double underscore "__".
